#  SmartDiet - Персональный AI-нутрициолог

> Сервис на базе **Yandex AI Studio**, который подбирает рецепты из продуктов в холодильнике с учётом медицинского профиля пользователя.

##  Проблема

Люди, которым необходимо соблюдать определенную диету, например из-за наличия хронических заболеваний (диабет, гипертония, гастрит), вынуждены постоянно продумывать каждый прием пищи. Сервис поможет решить проблему: AI знает ваш диагноз и строго соблюдает ограничения при создании каждого рецепта.

##  Возможности

- Медицинский профиль — диабет, гипертония, гастрит, лактоза, целиакия, подагра, аллергии
- Персонализированные рецепты с учётом заболеваний
- Расчёт нормы калорий по формуле Миффлина-Сан Жеора
- RAG по базе рецептов перед генерацией
- Проверка аллергенов на входе
- История питания между сессиями
- Статистика КБЖУ

## Запуск проекта 

### Предварительные требования
- Python 3.10+
- Аккаунт в Yandex Cloud с платежным аккаунтом
- Полученные API-ключи и Folder ID

### Получение ключей Yandex AI Studio
1. Зарегистрируйтесь в [Yandex Cloud]
2. Создайте платежный аккаунт 
3. Перейдите в [Yandex AI Studio]
4. Создайте API-ключ в разделе "Сервисные аккаунты"
5. Скопируйте Folder ID из консоли

###  Установка
\`\`\`bash
git clone https://github.com/your-team/smartdiet.git
cd smartdiet
pip install -r requirements.txt
\`\`\`

###  Настройка окружения
Создайте файл `.env` в корне проекта:
\`\`\`env
YANDEX_API_KEY=
YANDEX_FOLDER_ID=
AGENT_ID=
\`\`\`

###  Запуск
\`\`\`bash
streamlit run app.py
\`\`\`


##  Архитектура

```

app.py - Streamlit UI
gpt.py - Yandex AI Studio API
prompts.py - Системный промпт с медпрофилем
storage.py - JSON-хранилище профиля и истории
nutrition.py - База нутриентов, расчёт КБЖУ
rag.py - RAG-поиск по базе рецептов
recipes.json - База рецептов

```

### Поток данных

```
1 Ввод ингредиентов
2 Проверка аллергенов
3 RAG-поиск 
4 Сборка промпта (медпрофиль + ингредиенты + RAG) 
5 YandexGPT
6 Стриминг в UI
7 Сохранение в историю
```



##  Переменные окружения

| Переменная | Описание | Где взять |
|---|---|---|
| `YANDEX_API_KEY` | API-ключ сервисного аккаунта | Yandex Cloud → Сервисные аккаунты → API-ключи |
| `YANDEX_FOLDER_ID` | ID каталога | Yandex Cloud → шапка каталога |

##  Используемые возможности Yandex AI Studio

- **YandexGPT** — генерация рецептов
- **Агентный сценарий** — system prompt с медицинским контекстом
- **Персонализация** — медпрофиль в каждом запросе
- **RAG** — поиск по локальной базе рецептов
- **Структурированный вывод** — единый формат с КБЖУ

##  Ограничения

- Один пользователь на устройство (локальное хранение)
- База рецептов: 12 записей
- Нет мобильного приложения
